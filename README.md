# RecruitSync | Premium Resume Filtering Dashboard

A modern, fast, and responsive resume filtering dashboard designed for recruiters to screen, filter, and review candidates based on competencies, workforce segments, tools, and regions.

## Features

- **Dynamic Criteria-based Score:** Calculates a matching percentage for candidates based on matched requirements.
- **Advanced Filtering:** Sift through candidates by name, job title, and specific recruitment criteria (Competencies, Tools, Segments, Industry Specialization, and Geography).
- **Flexible Sorting:** Sort candidate listings by Match Score, Years of Experience, or Name.
- **Detailed Evaluation View:** View comprehensive candidate details, including email, phone, location, education, languages, and a structured criteria evaluation card with justifications.
- **Modern Responsive Design:** Crafted with a dark-themed glassmorphic UI using Vanilla CSS and CSS custom properties. It adapts smoothly to desktop, tablet, and mobile layouts.

## Tech Stack

- **Structure:** HTML5 (Semantic elements)
- **Styling:** Vanilla CSS (Custom variables, Flexbox, Grid, Media Queries)
- **Logic:** Vanilla JavaScript (ES6+)

## How to Run

Since this is a client-side web application, no installation or build steps are required. 
Simply open `index.html` in any modern web browser to run the application locally.
